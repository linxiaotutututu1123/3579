# 代码审查工作流

> **类型**: 辅助工作流 | **等级**: SUPREME

```yaml
工作流: CodeReviewWorkflow
参与Agent: 代码审查专家, 安全审计师, 测试工程师, 性能调优师
触发条件: PR提交/代码变更
```

## 审查流程

```
[自动检查] → [安全扫描] → [性能分析] → [人工审查] → [合并决策]
```

## 审查维度

```python
REVIEW_DIMENSIONS = {
    "代码质量": ["可读性", "可维护性", "重复度"],
    "安全性": ["注入", "XSS", "敏感数据"],
    "性能": ["复杂度", "资源使用"],
    "测试": ["覆盖率", "用例质量"],
    "合规": ["编码规范", "架构规范"],
}

APPROVAL_CRITERIA = {
    "自动通过": "所有检查通过+小变更",
    "需审批": "大变更/核心模块",
    "强制审批": "安全相关/架构变更",
}
```
