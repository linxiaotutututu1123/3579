# 知识沉淀工作流

> **类型**: 辅助工作流 | **等级**: SUPREME

```yaml
工作流: KnowledgePrecipitationWorkflow
参与Agent: 研究科学家, 文档专家, 量化架构师
触发条件: 项目完成/经验总结/知识更新
```

## 工作流程

```
[经验提取] → [知识整理] → [文档生成] → [知识验证] → [知识入库]
```

## 知识类型

```python
KNOWLEDGE_TYPES = {
    "策略知识": "策略设计经验",
    "技术知识": "技术实现方案",

    "故障知识": "故障处理经验",
}
```
